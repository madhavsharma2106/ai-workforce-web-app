# UX Philosophy

The user is a manager.

The employee is responsible for work.

The interface should feel calm.

## Language Mapping

Never expose AI terminology unless necessary.

| Replace this | With this  |
| ------------ | ---------- |
| Agent        | Employee   |
| Workflow     | Assignment |
| Prompt       | Work       |
| Execution    | Progress   |
| (LLM)        | (thinking) |

Users should never wonder what happens next.

This becomes incredibly useful when designing screens.
