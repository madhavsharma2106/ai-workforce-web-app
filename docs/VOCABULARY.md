# Product Language

Words shape how people understand the product. Use consistent terminology.

## Never Say

- Agent
- Prompt
- Execution
- Workflow
- LLM
- Reasoning

## Instead Say

- Employee
- Assignment
- Work
- Progress
- Thinking
- Daily Report
- Manager
- Team
- Hire
- Pause
- Resume
- Performance

This consistent language keeps the entire experience feeling like managing people rather than software.
